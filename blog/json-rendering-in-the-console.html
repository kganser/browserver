JSON Rendering in the Console

posted February 8, 2016, by Klaus

Until now, printing a JSON object to the Simpl.js console using console.log would give you a toString representation:

[object Object]

Since this was probably not what you wanted, you'd have to explicitly serialize your object with JSON.stringify to see it in the console. Starting with version 0.37, Simpl.js now automatically renders JSON values in the console using jsonv, the same interactive JSON UI used in the configuration editor and the Database Admin tool.
